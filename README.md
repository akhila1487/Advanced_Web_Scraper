# Web Scraping Job Data Project

## Objective
Scrape job listings and analyze hiring trends.

## Tools
- Python
- BeautifulSoup
- Pandas
- Matplotlib
- Seaborn

## Features
- Scraped job title, company, location
- Stored data in CSV
- Created visualization dashboard
- Analyzed top companies and locations

## Outcome
Built a real-world data scraping and analytics pipeline.
